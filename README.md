# Medical Cost Analysis — Excel Dashboard

## Project Overview
Analyzed medical insurance cost data for 1,300+ patients to identify 
key factors driving medical charges using Advanced Excel.

## Tools Used
- Advanced Excel (Pivot Tables, Slicers, Charts)
- Data Cleaning & Conditional Formatting
- Interactive Dashboard Design

## Key Insights
- Smokers pay significantly higher charges (avg $32,050)
- Obese patients have highest medical costs ($15,491)
- Northeast region has highest average charges ($13,406)
- Male and female charges are comparable ($13,957 vs $12,570)

## Dashboard Features
- 4 interactive Pivot Tables with Slicers
- Filters for Sex, Smoker status, Region, BMI Classification
- Bar charts, pie chart for visual comparison of cost drivers
